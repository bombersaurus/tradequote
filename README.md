# TradeQuote

TradeQuote is a focused MVP for UK electricians to create professional quotes quickly, preview the customer-facing version, and export a PDF.

## What is included

- Electrician-specific quote templates.
- Customer, job, VAT, deposit, notes, and line item editing.
- Live quote preview.
- PDF export.
- Saved reusable line items in local storage.
- Early-access email CTA for founder customers.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy

Deploy on Vercel with the default Next.js settings.

Optional environment variable:

```bash
NEXT_PUBLIC_CONTACT_EMAIL=you@example.com
```

If it is not set, the app uses `hello@tradequote.app` as a placeholder contact email.

## First customer positioning

TradeQuote helps electricians create clean, professional quotes in minutes with VAT, deposits, line items, and PDF export.
